# VAST Data Demos & Tools

A collection of demos, dashboards, and utilities for [VAST Data](https://vastdata.com) platform environments.

## Projects

| Project | Description |
|---------|-------------|
| [**audit-log-dashboards**](audit-log-dashboards/) | Pre-built Grafana dashboards for VAST audit log analytics via Trino. Includes overview, file lineage tracking, hot file detection, and per-user IO breakdown. Ships as a single Docker container — just point it at your Trino endpoint. |
